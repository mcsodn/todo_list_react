# Приложение «Список задач»

Реализовано с использованием: Vite, React, JSX и Tailwind CSS.

## Детали

- в приложении выделено 4 компонента
- использованы хуки useState, useEffect и useRef
- добавлен хук useLocalStorage для хранения данных между перезагрузками
- применен пакет react-swipeable для мобильных

Проект опубликован по адресу: https://mcsodn-tdr.netlify.app

## Локальная сборка

После перехода в папку проекта нужно запустить команду

> yarn install; yarn run dev
